## ⚠️ Prérequis

- Python **3.11 uniquement**
- Windows / Linux OK
- pygame compatible uniquement en 3.11

---

## 🚀 Installation (ULTRA SIMPLE)

```bash
git clone <repo_url>
cd theorie-des-jeux
make setup
make run
```

Les fichiers du jeu sont maintenant directement à la racine du dépôt.
